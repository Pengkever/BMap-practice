# BMap-practice

## 使用方法
* `clone`到本地，在浏览器中打开`index.html`文件。
* ~~修改`js/app.js`文件中`load(x, y)`的参数值，来获取更多内容~~（已经新增）
* 从百度行政区域搜索api中获取了10条相关内容，在输入框中输入条件内容，进行筛选
* 可以直接点击列表中的你喜欢的项目
* 点击`back`按钮返回列表
